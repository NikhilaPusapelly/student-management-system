# Student Management System (Tkinter)

A desktop application built using **Python Tkinter** that allows students and administrators to manage student records.

## Features

* Student Login
* Admin Login Dashboard
* Create Student Account
* SQLite Database Storage
* Student Card Generator
* Email Notification System

## Tech Stack

* Python
* Tkinter
* SQLite
* Pillow

## How to Run

Clone the repository:

git clone https://github.com/NikhilaPusapelly/student-management-system.git

Install dependencies:

pip install pillow pywin32

Run the project:

python main.py
